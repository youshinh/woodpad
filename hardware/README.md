# 筐体設計ドキュメント - WoodPad

## 1. コンセプト
「木材の質感」と「高度なデジタル操作性」を融合させた、ステルスデザインのBluetoothトラックパッド。

## 2. デザイン方針
* **ハイブリッド構造:**
    * 外装: 無垢材（木工）による温かみと安定感。
    * 内部保持: 3Dプリンターによる0.1mm単位で設計された精密パーツ。
* **ステルス設計:** センサーを木材の裏に隠すことで、操作面をシームレスにする。

## 3. 主要構成パーツ
* **木材トッププレート:** ステルス操作面用（厚み：試作検討）。
* **3Dプリント保持パーツ:** センサー固定用。
* **土台筐体:** 木材削り出しまたは積層。
